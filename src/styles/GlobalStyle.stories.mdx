# Global Style

CSS Resets stylesheets

```jsx
const App = () => (
  <>
    <GlobalStyle />
    <OtherComponent />
  </>
);
 `;
```

## Typography

Basic global typography.

All HTM headings. `h1` through `h6`, are available:

  <Story name="heading">
    <div>

  </Story>
</Preview>
