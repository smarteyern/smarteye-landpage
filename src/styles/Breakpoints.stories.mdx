# Breakpoints

Breakpoints for responsives layout:

<table>
  <tr>
    <th>Extra Small (xs)</th>
    <td>Default | menor que {BreadkpointsSize.xs}px</td>
  </tr>
  <tr>
    <th>Small (sm)</th>
    <td> >= {BreadkpointsSize.sm}px</td>
  </tr>
  <tr>
    <th>Medium (md)</th>
    <td> >= {BreadkpointsSize.md}px</td>
  </tr>
  <tr>
    <th>Large (lg)</th>
    <td> >= {BreadkpointsSize.lg}px</td>
  </tr>
  <tr>
    <th>Extra large (xl)</th>
    <td> >= {BreadkpointsSize.xl}px</td>
  </tr>
  <tr>
    <th>Extra extra large (xxl)</th>
    <td> >= {BreadkpointsSize.xxl}px</td>
  </tr>
</table>

```js
import { breakAt, BreadkpointsSize } from "Breackpoints";

const title = styled.h1`
  font-size: 1rem;
  ${breakAt(BreadkpointsSize.md)} {
    font-size: 2rem;
  }
`;
```
